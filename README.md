Para que o código funcione corretamente, você precisa fazer o upload da base de dados para dentro do ambiente virtual do Colab antes de executá-lo. Siga este passo a passo:

Abra o link do projeto no Google Colab.

No menu lateral esquerdo da tela, clique no ícone de uma Pasta (chamado "Arquivos").

Por padrão, o Colab já abre na pasta principal de trabalho, que é o diretório /content/.

Para adicionar os arquivos, clique no ícone de Upload (uma folhinha com uma seta para cima) que fica logo abaixo da palavra "Arquivos".

Selecione no seu computador os arquivos perguntas.csv e respostas.json e faça o upload. (Se preferir, você também pode simplesmente arrastar os dois arquivos do seu computador e soltar dentro dessa aba lateral).

Aguarde o carregamento. Quando os nomes dos dois arquivos aparecerem na lista abaixo da pasta sample_data, o ambiente estará pronto!

Agora é só rodar o código normalmente clicando no botão de "Play" das células ou em "Ambiente de execução" > "Executar tudo".
